## Kubernetes Plugins

| Plugin Category          | Examples                         |
|--------------------------|---------------------------------|
| **CNI Plugins**         | Calico, Flannel, Cilium        |
| **CSI Plugins**         | AWS EBS, OpenEBS, Longhorn     |
| **CRI Plugins**         | containerd, CRI-O              |
| **CoreDNS Plugins**     | kubernetes, forward, cache     |
| **Kube Proxy Modes**    | iptables, IPVS, eBPF           |
| **Admission Controllers** | ValidatingWebhook, ResourceQuota |
| **Volume Plugins**      | NFS, CephFS, iSCSI             |
| **Ingress Controllers** | NGINX, Traefik, AWS ALB        |
| **Scheduler Plugins**   | Default Scheduler, Descheduler |
| **CRDs & Operators**    | Prometheus, Istio, Kafka       |
